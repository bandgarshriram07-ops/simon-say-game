# javaScript Project Useing DOM
 Simon Says Game / シモンゲーム

A simple memory game where players repeat color sequences.
色の順番を覚えて繰り返すシンプルな記憶ゲームです。

#Features / 特徴
Interactive gameplay / インタラクティブなゲーム
Increasing difficulty / レベルが徐々に上がる
Game over detection / ミスするとゲーム終

#How to Play / 遊び方
1 Press any key to start
　キーを押して開始
2 Watch the color sequence
　色の順番を覚える
3 Repeat the sequence
　同じ順番でクリック
4 Wrong click = Game Over
　間違えるとゲームオーバー
